# 🎮 Sprint 1 - Recomendador de Videojuegos

## 📋 Descripción
Sistema de recomendación de videojuegos desarrollado en Python como parte de la materia 
Programación 1 (Curso 211). El programa solicita datos al usuario y mediante reglas 
condicionales combinadas le brinda la mejor recomendación de videojuego según sus necesidades.

## 🎯 Objetivo
Desarrollar un sistema de recomendación que evalúe múltiples variables mediante reglas 
condicionales combinadas, procesando datos del usuario para sugerir el videojuego más adecuado.

## 🚀 ¿Cómo se usa?
1. Ejecutar el archivo `sprint1.py`
2. Ingresar los datos que el programa solicita:
   - Nick de jugador
   - Mail
   - Plataforma (Playstation / PC / Mobile)
   - Formato (Fisico / Digital)
   - Género (Terror / Accion / Aventura)
   - Edad
   - Capacidad de almacenamiento en GB
   - Presupuesto
   - Cantidad de jugadores
   - Cantidad de juegos que desea comprar
3. Aceptar los términos y condiciones
4. El programa mostrará la recomendación más adecuada según tus datos
